CRESHCHAT DUNGEON CRAWLER - COMPLETE CORE ASSET PACK
Version 4.0 - WoW TBC Anniversary / Classic Asset Package

LIVE ASSETS
- 20 expansion enemies, each with a 128x128 icon and 256x512 full-body texture.
- 40 class armor sets converted into live textures.
- Current complete classes: Paladin, Warrior, Rogue, Ranger, Mage, Priest, Warlock, Defender.
- 5 milestone chests, each with a 128x128 icon and 512x512 display texture.
- 5 reward icons converted into live textures.

FORMAT
- TGA, 32-bit RGBA with alpha.
- Icons: 128x128.
- Class full body: 256x512.
- Chest display: 512x512.
- Power-of-two dimensions suitable for WoW Classic/TBC addon textures.

INSTALL
1. Extract this ZIP.
2. Copy the included Media folder into the CreshChat addon root and merge folders.
3. Add Code/DungeonCrawlerAssetLibrary.lua to the addon TOC before files that reference these paths, or use Docs manifests to wire the paths into the existing data tables.
4. WoW Lua texture references should omit the .tga extension. The supplied Lua map already does so.
