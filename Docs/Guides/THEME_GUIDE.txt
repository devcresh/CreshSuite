NOTIFICATION APPEARANCE

ZLR ARENA
- Idle launcher and console logo text: q3a.
- Composer prompt: /say or /console.
- Notification letters temporarily replace q3a and restore to q3a when cleared.


THEME PREVIEW WORKFLOW
- Settings > Themes stores the dropdown choice only in the active Settings session.
- PREVIEW snapshots the equipped theme and seven global surface colours, then applies the selected preset through the normal visual refresh path.
- Locked themes may be previewed, but APPLY THEME still uses BattlePass:IsThemeUnlocked before committing.
- REVERT and Settings OnHide restore the original preset and colour snapshot.
- Preview state is never written to SavedVariables.
