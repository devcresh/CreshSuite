## Native messaging aliases

CreshChat routes these directly through the compatible WoW chat APIs:

- `/w`, `/whisper`, `/tell`, `/t`, `/send` and `/r`, `/reply`
- `/g`, `/guild`, `/o`, `/officer`, `/p`, `/party`
- `/raid`, `/ra`, `/rsay`, `/rw`, `/raidwarning`
- `/i`, `/instance`, `/bg`, `/battleground`
- `/s`, `/say`, `/y`, `/yell`, `/e`, `/emote`
- Numbered channels such as `/1`, `/2` and `/4`

All other slash commands continue through Blizzard's native command handler.
