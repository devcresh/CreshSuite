CreshChat v0.2 - Final Compatibility and UI Audit

- Reset the public addon version to v0.2 while retaining SavedVariables schema 76 so existing account-wide progress is preserved.
- Added responsive main-header layout; hidden whisper controls no longer reserve space or crush the title at narrow console widths.
- Added missing live settings for default composer destination, roster collapse, history limits, build badge, compact notification sizing, card accent thickness and dock-whisper preview sizing/timing.

- Reduced addon-message traffic: peer discovery is limited, paced and cooldown-protected; login no longer probes the full social roster; live game snapshots update at safer rates.

CreshChat v0.3.53-dungeon-armour-stats

Dungeon Dweller armour loadouts
- Converted all 50 class armour skins from cosmetic records into distinct gameplay loadouts.
- Added armour bonuses for Max HP, Attack, fixed extra dice, chance-based bonus dice, double damage, bleed, turn regeneration, room healing, block, evasion, boss damage and minion damage.
- Added a five-card armour selection window for the active Dungeon class, including final placeholder artwork, ownership state, stat comparison and one-click equip.
- Added an ARMOUR button beside CLASS and the V keyboard shortcut.
- Equipping or removing armour starts a fresh run so combat statistics cannot be changed during a boss attempt.
- Equipped armour now replaces the hero icon/full-body artwork and is shown in the hero status panel.
- Extra armour dice roll independently; secondary dice use the set's listed power scaling.
- Bleed effects persist on enemies and tick before their next attack.
- Health regeneration activates after enemy turns and/or cleared rooms according to the equipped set.
- Defensive armour statistics stack with class block and evasion, with safety caps to prevent immunity.
- The first armour unlocked for a class is automatically selected for the next run; later sets remain manual choices.
- Migrated SavedVariables to schema 54 without resetting boss kills, armour ownership, equipped sets, crates, Tetris data, Cresh Coins or character profiles.

The level-10 milestone boss system, 20 bosses, 20 normal enemy additions, four crate types and all placeholder artwork remain included.
